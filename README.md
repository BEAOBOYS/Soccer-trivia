# Soccer Stat Encoder ⚽

A JavaScript console project built for CodeHS that collects soccer player statistics and demonstrates how text is encoded into ASCII and binary.

## Project Description
Soccer Stat Encoder allows users to enter player names and goals scored, stores the data using objects and arrays, categorizes player performance, and encodes player names into ASCII and binary.

## How to Run
1. Open CodeHS
2. Create a JavaScript Console project
3. Paste the contents of soccerStatEncoder.js
4. Run the program
