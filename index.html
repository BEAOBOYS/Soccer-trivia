⚽ Soccer Stat Encoder

Welcome! This project is a JavaScript console program designed for the CodeHS JavaScript Console. It collects soccer player statistics and demonstrates how text is encoded using ASCII and binary.

📌 Project Overview

Users enter soccer player names and the number of goals scored. The program stores each player as an object, categorizes performance, and converts names into ASCII and binary values to demonstrate how computers store text data.

🧠 Computer Science Concepts

  • Functions with parameters
  • Arrays and objects
  • Loops and conditionals
  • ASCII and binary encoding
  • User input and console output

▶️ How to Run

  1. Open CodeHS
  2. Create a JavaScript Console project
  3. Copy the contents of soccerStatEncoder.js
  4. Run the program and follow the prompts

🔧 Possible Extensions

  • Calculate average goals scored
  • Add a player search feature
  • Decode binary back into text

This project is intended for educational use and fits within one class period.
